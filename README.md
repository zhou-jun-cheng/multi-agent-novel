# 📖 AI 小说创作机器人

> 多智能体协作，自动完成小说创作全流程

一个基于 DeepSeek AI 的自动化小说创作系统。通过 8 个各具特色的 AI 机器人协作，从大纲规划、正文写作到多维度审核，实现完整的小说章节自动生成。

## ✨ 特性

- **8 个 AI 机器人协作** — 每个机器人有独特角色定位和人格设定
- **审核-反馈循环** — 四维审核（剧情/文笔/人物/节奏），得分低于 80 自动打回重写
- **上下文连贯** — 自动管理前后章节内容、人物性格、人物关系
- **一键续写/新写** — 支持续写已有小说或开始全新创作

## 🤖 机器人介绍

| 机器人 | 角色 | 职责 |
|--------|------|------|
| 墨章 | 章节规划师 | 首次创作时规划整本书的章节结构 |
| 墨云 | 总策划 | 设计章节大纲、世界观、人物关系 |
| 墨建 | 大纲评审 | 审核大纲质量，打分决定是否通过 |
| 墨笔 | 正文写手 | 根据大纲撰写 2500+ 字正文 |
| 司命 | 剧情审核 | 审核剧情逻辑与连贯性 |
| 墨文 | 文笔审核 | 审核文字质量与表达 |
| 知人 | 人物审核 | 审核人物塑造与性格一致性 |
| 律动 | 节奏审核 | 审核叙事节奏与张力 |
| 墨结 | 总结者 | 生成章节总结、更新人物关系与剧情概要 |

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key

复制环境变量模板并填入你的 DeepSeek API Key：

```bash
cp .env.example .env
```

然后在 `.env` 文件中设置：

```
API_KEY=your_deepseek_api_key_here
```

> 获取 API Key: https://platform.deepseek.com

### 3. 运行

```bash
python start.py
```

根据提示选择：
- `yes` — 续写已有小说（需输入小说名称）
- `no` — 开始全新创作

## ⚙️ 工作原理

```
墨章规划全书章节
       ↓
   [循环开始]
       ↓
  墨云 → 生成章节大纲
       ↓
  墨建 → 评审大纲（≤80分打回重写）
       ↓
  墨笔 → 撰写正文（≥2200字）
       ↓
  审核组 → 四维审查（司命/墨文/知人/律动）
       ↓
  全部通过？
  ├─ 是 → 墨结总结 → 保存 → [下一章]
  └─ 否 → 打回墨笔重写
```
